# Padh.AI Development Commands

## 🚀 Build & Run Commands

### 1. Start Metro Server
Open a dedicated terminal and run:
```bash
npm start
```

### 2. Run Debug Version (on Device/Emulator)
Ensure your device is connected via ADB, then run:
```bash
npx react-native run-android
```

### 2a. Build & Install Debug APK (Alternative Method)
Build debug APK and install it manually:
```bash
cd android && ./gradlew installDebug
```
Or build first, then install:
```bash
cd android && ./gradlew assembleDebug
adb install app/build/outputs/apk/debug/app-debug.apk
```

### 3. Build Standalone Release APK
To generate a production-ready, standalone APK file:
```bash
cd android && ./gradlew assembleRelease
```
**TIP:** The generated APK will be located at: `android/app/build/outputs/apk/release/app-release.apk`

### 4. Build & Install Release (Direct to Device)
If you want to test the release version directly on your device:
```bash
npx react-native run-android --mode release
```
**Note:** If you encounter a Permission Denied error when running the Gradle command, run `chmod +x android/gradlew` first.

### 5. Clean Build (Fix Build Issues)
```bash
cd android && ./gradlew clean
cd .. && npm start -- --reset-cache
```

### 6. Install Dependencies
```bash
npm install
cd android && ./gradlew --refresh-dependencies
```

## 📱 Device Management

### List connected devices:
```bash
adb devices
```

### Restart ADB server (fixes connection issues):
```bash
adb kill-server && adb start-server
```

### Connect to device over WiFi:
```bash
# First connect via USB, then:
adb tcpip 5555
adb connect <DEVICE_IP>:5555
```

### Check device info:
```bash
adb shell getprop ro.product.model
adb shell getprop ro.build.version.release
```

## 🔧 React Native Specific

### Reverse Port Forwarding (connects device to Metro server):
```bash
adb reverse tcp:8081 tcp:8081
```

### Open Developer Menu on the device:
```bash
adb shell input keyevent 82
```

### Reload the app:
```bash
adb shell input keyevent 46 46
```

### Enable/Disable Fast Refresh:
```bash
# Enable
adb shell input keyevent 82
# Then tap "Enable Fast Refresh"

# Disable
adb shell input keyevent 82
# Then tap "Disable Fast Refresh"
```

## 📁 File & System

### Take a screenshot:
```bash
adb shell screencap -p /sdcard/screen.png && adb pull /sdcard/screen.png .
```

### Check Model storage directory (where we save AI models):
```bash
adb shell ls /storage/emulated/0/Android/data/com.padhai/files/models
```

### Pull model file from device:
```bash
adb pull /storage/emulated/0/Android/data/com.padhai/files/models/model.task .
```

### Push model file to device:
```bash
adb push model.task /storage/emulated/0/Android/data/com.padhai/files/models/
```

### Check app storage size:
```bash
adb shell du -sh /storage/emulated/0/Android/data/com.padhai
```

### View app database:
```bash
adb shell run-as com.padhai ls /data/data/com.padhai/databases
```

### Pull database for inspection:
```bash
adb shell run-as com.padhai cat /data/data/com.padhai/databases/padh.db > padh.db
```

## 🔐 Signing & Release

### Generate keystore:
```bash
keytool -genkeypair -v -storetype PKCS12 -keystore padhai-release-key.keystore -alias padhai-key-alias -keyalg RSA -keysize 2048 -validity 10000
```

### Check keystore:
```bash
keytool -list -v -keystore padhai-release-key.keystore
```

### Sign APK manually:
```bash
jarsigner -verbose -sigalg SHA256withRSA -digestalg SHA-256 -keystore padhai-release-key.keystore app-release-unsigned.apk padhai-key-alias
```

### Verify APK signature:
```bash
jarsigner -verify -verbose -certs app-release.apk
```

## 🎯 Quick Troubleshooting

### App won't start:
```bash
adb uninstall com.padhai
npm start -- --reset-cache
npx react-native run-android
```

### Metro bundler issues:
```bash
pkill -f "node.*metro"
npm start -- --reset-cache
```

### Gradle build fails:
```bash
cd android && ./gradlew clean
cd .. && rm -rf android/.gradle
cd android && ./gradlew assembleDebug
```

### Native module not found:
```bash
cd android && ./gradlew clean
cd .. && npx react-native run-android
```

### Device not detected:
```bash
adb kill-server
adb start-server
adb devices
```

## 💡 Tips

**Multiple Devices:** If you have multiple devices connected, specify the device ID using `-s`:
```bash
adb -s <DEVICE_ID> reverse tcp:8081 tcp:8081
```

**Find Device ID:**
```bash
adb devices -l
```

**Check React Native version:**
```bash
npx react-native --version
```

**Update React Native:**
```bash
npx react-native upgrade
```

**Check Node version:**
```bash
node --version
```

**Check npm version:**
```bash
npm --version
```
