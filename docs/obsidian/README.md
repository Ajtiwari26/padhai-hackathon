# Padh.ai Project Vault 🧠

This directory is designed to be opened as an **Obsidian Vault**. It contains all the technical documentation, design decisions, and AI-assisted notes for the Padh.ai project.

## Setup Instructions

1.  **Install Obsidian**: Download from [obsidian.md](https://obsidian.md/).
2.  **Open Vault**: In Obsidian, select "Open folder as vault" and choose this directory (`docs/obsidian`).
3.  **Install Claude Code Integration**:
    *   Install the **BRAT** plugin from Community Plugins.
    *   Add the beta plugin: `https://github.com/deivid11/obsidian-claude-code-plugin`.
    *   Enable "Claude Code Integration".
4.  **Configure**:
    *   Set the path to your `claude` or `antigravity` CLI.
    *   The AI will now have context of your notes and can help you maintain them.

## Vault Structure

- `DESIGN.md`: Anchor for the design system and architecture.
- `RESEARCH.md`: Link to tool research and integration plans.
- `TASKS.md`: Project roadmap and active task lists.
- `CONCEPTS/`: Directory for deep dives into specific AI/Mobile concepts.

## Efficiency Tools

- **Caveman Mode**: Toggle with `/caveman` to save 65% tokens on documentation updates.
- **CRG Integration**: Visualized code dependencies for better architecture notes.
